# TypeScript
1. 静态类型: 声明一个变量要同时指定它的类型
2. 自带类型推断: 不需要指定类型，会根据值自动推断类型

# TS 类型
- any: 任意类型，不推荐使用
- unknow: 未知类型，推荐使用
- never: 永远类型，不会存在的值的类型
- js 中的类型: 
    - string     字符串      String(既可以是字面量也可以是对象)
    - number     数字        Number
    - boolean    布尔        Boolean
    - null       空         
    - undefined  未定义      
    - symbol     符号        Symbol
    - bigint     大整数      BigInt

    - Object     广义对象类型 约等于 any
    - object     狭义对象类型
